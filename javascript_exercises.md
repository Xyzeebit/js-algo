# 30 JavaScript Exercises for Beginners

1. **Temperature Converter** – Write functions to convert Celsius to Fahrenheit and vice versa.

2. **Factorial (Recursive)** – Compute the factorial of a given number using recursion.

3. **Fibonacci Sequence** – Generate the first `n` numbers of the Fibonacci sequence.

4. **Palindrome Checker** – Determine if a string is a palindrome (ignore spaces and punctuation).

5. **FizzBuzz** – Print numbers 1 to 100; for multiples of 3 print "Fizz", for multiples of 5 print "Buzz", and for both print "FizzBuzz".

6. **Reverse a String** – Return the reversed version of a given string.

7. **Count Vowels** – Count the number of vowels (a, e, i, o, u) in a string.

8. **Find Maximum in Array** – Write a function that returns the largest number in an array.

9. **Sum of Array** – Calculate the sum of all numbers in an array.

10. **Filter Even Numbers** – From an array of numbers, return a new array containing only the even numbers.

11. **Map to Squares** – Transform an array of numbers by squaring each element.

12. **Array Product (Reduce)** – Use `reduce` to compute the product of all numbers in an array.

13. **Student Records (Object Array)** – Create an array of student objects (name, age, grade) and write a function to find students with a grade above a given threshold.

14. **Sort by Age** – Sort an array of people objects by age in ascending order.

15. **Simple Calculator** – Implement `add`, `subtract`, `multiply`, `divide` functions and a `calculate(operator, a, b)` that uses them.

16. **Closure Counter** – Write `createCounter` that returns a function which increments and returns a counter on each call.

17. **Delayed Logging** – Use `setTimeout` to print numbers 1 through 5 with a one-second delay between each.

18. **Promise Delay** – Create a function `delay(ms)` that returns a promise which resolves after `ms` milliseconds.

19. **Fetch API Data (Simulated)** – Write an async function that fetches user data from a public API (e.g., JSONPlaceholder) and logs the names.

20. **Debounce Function** – Implement a debounce utility that delays invoking a function until after a wait time.

21. **Throttle Function** – Implement a throttle utility that ensures a function is called at most once in a specified interval.

22. **Deep Clone Object** – Write a function that deep clones an object (handles nested objects and arrays).

23. **Flatten Nested Array** – Convert a nested array (e.g., `[1, [2, 3], 4]`) into a flat array.

24. **Array Chunking** – Split an array into smaller arrays of a specified size.

25. **Anagram Check** – Determine if two strings are anagrams (contain the same characters in any order).

26. **Unique Values** – Return a new array containing only the unique values from an input array.

27. **Implement Map/Filter/Reduce** – Write your own versions of `map`, `filter`, and `reduce` as standalone functions.

28. **Memoized Fibonacci** – Optimize the Fibonacci function using memoization to avoid repeated calculations.

29. **Simple Event Emitter** – Create an `EventEmitter` class with `on`, `emit`, and `off` methods.

30. **Singly Linked List** – Implement a linked list with methods: `append`, `prepend`, `delete`, and `display`.
